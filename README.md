# 🏥 Hospital Healthcare Analytics Dashboard

## 📌 Short Description
The **Hospital / Healthcare Analytics Dashboard** is a data analysis and visualization project built using **SQL and Power BI**.

This project transforms raw hospital data into meaningful insights to understand:
- Patient trends  
- Revenue generation  
- Operational efficiency  
- Resource utilization  

It provides a centralized view of hospital performance and supports **data-driven decision-making**.

---

## 🎯 Project Objectives
- Analyze patient admission patterns over time  
- Evaluate hospital revenue across departments  
- Monitor average patient stay duration  
- Understand disease distribution and treatment demand  
- Assess doctor performance and workload  
- Analyze insurance coverage impact on revenue  

---

## ⚙️ Tech Stack
- **SQL (MySQL)** – Data extraction & analysis  
- **Power BI** – Dashboard creation & visualization  
- **DAX** – KPI calculations & measures  
- **CSV / Excel** – Data storage  

---

## 📂 Data Source
- **Dataset Type:** Synthetic Hospital Dataset  
- **Format:** CSV  
- **Total Records:** 3000+  

---

## ✨ Features / Highlights

### 📊 KPI Metrics
- Total Patients  
- Total Revenue  
- Average Stay Duration  
- Insurance Coverage %  

---

### 📈 Data Visualization
- Monthly Admissions Trend (Line Chart)  
- Department-wise Revenue (Bar Chart)  
- Gender Distribution (Pie Chart)  
- Disease Analysis (Bar Chart)  
- Doctor Performance (Table)  

---

### 🧠 SQL Analysis
- Patient & revenue aggregation  
- Department performance analysis  
- Disease frequency tracking  
- Monthly trend analysis  
- Used `GROUP BY`, `JOIN`, and aggregate functions  

---

### 🔍 Interactive Dashboard
- Filters (Slicers):
  - Department  
  - Gender  
  - Date Range  
  - Insurance Status  
- Real-time dynamic filtering  

---

### 🔄 Data Processing
- Cleaned and transformed raw CSV data  
- Created:
  - Stay Duration (Admit → Discharge)  
  - Month column for trend analysis  

---

### 📐 DAX Calculations
- Total Revenue  
- Average Stay  
- Insurance %  

---

### 💡 Business Insights
- High revenue-generating departments  
- Seasonal admission trends  
- Doctor workload distribution  
- Common diseases

### 🎯 Business Impact
This dashboard helps:
