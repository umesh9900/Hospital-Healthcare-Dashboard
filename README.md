# Hospital-Healthcare-Dashboard
## Recommended Structure and Order
### 1.	Project Title
The Hospital / Healthcare Analytics Dashboard is a data analysis and visualization project built using SQL and Power BI. This project focuses on transforming raw hospital data into meaningful insights that help in understanding patient trends, revenue generation, operational efficiency, and resource utilization.
The dashboard provides a centralized view of hospital performance and supports data-driven decision-making for management.

### 2.	Short Description
The main objectives of this project are:

To analyze patient admission patterns over time
To evaluate hospital revenue across departments
To monitor average patient stay duration
To understand disease distribution and treatment demand
To assess doctor performance and workload
To analyze insurance coverage impact on hospital revenue

This project simulates a real-world healthcare analytics use case.
